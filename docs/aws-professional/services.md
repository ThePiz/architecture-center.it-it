# <a name="aws-to-azure-services-comparison"></a>Confronto tra i servizi di Azure e AWS

Questo articolo aiuta a identificare le differenze tra i servizi di Microsoft Azure e Amazon Web Services (AWS). Quando si pianifica una soluzione multi-cloud con Azure e AWS oppure si esegue la migrazione in Azure, è possibile confrontare le funzionalità IT dei servizi di Azure e AWS in tutte le categorie.

Nelle tabelle seguenti sono elencati più servizi di Azure per alcuni servizi AWS. I servizi di Azure sono simili tra loro, ma la completezza e l'ampiezza delle funzionalità possono variare.

## <a name="azure-and-aws-for-multicloud-solutions"></a>Azure e AWS per soluzioni multi-cloud

Microsoft Azure e Amazon Web Services (AWS), le piattaforme su cloud pubblico leader del settore, offrono alle aziende un set ampio e completo di funzionalità con copertura globale. Tuttavia, molte organizzazioni scelgono di usare entrambe le piattaforme in insieme, per ottenere scelta e flessibilità maggiori e per limitare i rischi e le dipendenze grazie a un approccio multi-cloud. Anche le società di consulenza e i fornitori di software possono scegliere di usare le piattaforme Azure e AWS insieme, in quanto sono in grado di soddisfare la maggior parte della domanda del mercato cloud.

Per una panoramica di Azure per gli utenti di AWS, vedere l'[introduzione ad Azure per i professionisti AWS](./index.md).
