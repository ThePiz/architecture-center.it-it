## <a name="machine-learning"></a><span data-ttu-id="10070-132">Apprendimento automatico</span><span class="sxs-lookup"><span data-stu-id="10070-132">Machine learning</span></span>
<span data-ttu-id="10070-133">L'apprendimento automatico è una tecnica di modellazione matematica usata per eseguire il training di un modello predittivo.</span><span class="sxs-lookup"><span data-stu-id="10070-133">Machine learning is a mathematical modeling technique used to train a predictive model.</span></span> <span data-ttu-id="10070-134">Il principio generale consiste nell'applicazione di un algoritmo statistico a un ampio set di dati cronologici per individuare le relazioni tra i vari campi.</span><span class="sxs-lookup"><span data-stu-id="10070-134">The general principle is to apply a statistical algorithm to a large dataset of historical data to uncover relationships between the fields it contains.</span></span>

<span data-ttu-id="10070-135">La modellazione dell'apprendimento automatico viene normalmente eseguita da data scientist, che devono esaminare attentamente i dati e prepararli prima di eseguire il training di un modello.</span><span class="sxs-lookup"><span data-stu-id="10070-135">Machine learning modeling is usually performed by data scientists, who need to thoroughly explore and prepare the data before training a model.</span></span> <span data-ttu-id="10070-136">L'esplorazione e la preparazione dei dati richiedono in genere molte attività interattive di visualizzazione e analisi, che vengono eseguite tramite linguaggi come Python e R in strumenti e ambienti interattivi, appositamente progettati per queste attività.</span><span class="sxs-lookup"><span data-stu-id="10070-136">This exploration and preparation typically involves a great deal of interactive data analysis and visualization &mdash; usually using languages such as Python and R in interactive tools and environments that are specifically designed for this task.</span></span>

<span data-ttu-id="10070-137">In alcuni casi possono essere disponibili [modelli sottoposti a training preliminare](/machine-learning-server/install/microsoftml-install-pretrained-models), che contengono dati di training ottenuti e sviluppati da Microsoft.</span><span class="sxs-lookup"><span data-stu-id="10070-137">In some cases, you may be able to use [pretrained models](/machine-learning-server/install/microsoftml-install-pretrained-models) that come with training data obtained and developed by Microsoft.</span></span> <span data-ttu-id="10070-138">Questi modelli offrono il vantaggio di poter assegnare un punteggio a nuovi contenuti e classificarli immediatamente, anche se non si hanno i dati di training necessari, le risorse per gestire grandi set di dati o eseguire il training di modelli complessi.</span><span class="sxs-lookup"><span data-stu-id="10070-138">The advantage of pretrained models is that you can score and classify new content right away, even if you don't have the necessary training data, the resources to manage large datasets or to train complex models.</span></span>

<span data-ttu-id="10070-139">Esistono due tipologie generali di apprendimento automatico:</span><span class="sxs-lookup"><span data-stu-id="10070-139">There are two broad categories of machine learning:</span></span>

* <span data-ttu-id="10070-140">**Apprendimento supervisionato**.</span><span class="sxs-lookup"><span data-stu-id="10070-140">**Supervised learning**.</span></span> <span data-ttu-id="10070-141">Questo è l'approccio più diffuso per l'apprendimento automatico.</span><span class="sxs-lookup"><span data-stu-id="10070-141">Supervised learning is the most common approach apprendimento automatico, il set di dati include sia le caratteristiche sia le etichette note e viene applicato un algoritmo per modellare una funzione che agisce sulle caratteristiche per calcolare * <span data-ttu-id="10070-146">**Apprendimento non supervisionato**.</span><span class="sxs-lookup"><span data-stu-id="10070-146">**Unsupervised learning**.</span></span> <span data-ttu-id="10070-147">In un modello di apprendimento non supervisionato i dati di training non includono valori di etichetta noti.</span><span class="sxs-lookup"><span data-stu-id="10070-147">In an unsupervised learning model, the training data does not include known label values.</span></span> <span data-ttu-id="10070-148">Diversamente da quanto avviene nell'altro modello, l'algoritmo esegue le stime in base alla prima esposizione ai dati.</span><span class="sxs-lookup"><span data-stu-id="10070-148">Instead, the algorithm makes its predictions based on its first exposure to the data.</span></span> <span data-ttu-id="10070-149">La forma più comune di apprendimento non supervisionato è definita *clustering*.
